### Setup & Launch

#### Vue.js Launch
1. `npm install` (project setup)
2. `npm run dev` (compile and hot-reload for development @ http://localhost:3000)
3. `npm run build` (type-check, compile and minify for production)
4. `npm run lint` (lint with ESLint)
